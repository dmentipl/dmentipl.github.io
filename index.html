About

I am a computational astrophysics PhD student at the School of Physics and Astronomy at Monash University. I use numerical simulations to study protoplanetary discs—the birthplace of planets.

Research

I work with Daniel Price and Christophe Pinte on hydrodynamical and radiative transfer modelling of protoplanetary discs.

We perform global dust-gas smoothed particle hydrodynamics simulations of protoplanetary discs with embedded planets using Phantom, and post-process these simulations with the Monte Carlo radiative transfer code MCFOST.

Publications

I have applied these techniques, for example, to model the disc around TW Hya, and to study the effect of stellar flybys on protoplanetary discs in work led by Nicolas Cuello.

My full publication list is available on ORCID.

Data

I make datasets relating to my work available on figshare.

Of particular interest may be the two datasets relating to the TW Hya work: one contains output from Phantom models, and the other contains output from MCFOST post-processed models.

Software

I write software in Python and (modern) Fortran.

For example, I am the main author of Plonk, a smoothed particle hydrodynamics analysis and visualization tool written in Python. See the paper at JOSS.

Other Python packages for working with Phantom include:

  • phantom-config — read, modify, convert, and write Phantom configuration files;

  • phantom-build — make it easy to generate reproducible Phantom builds for writing reproducible papers;

  • phantom-setup — generate initial conditions for Phantom simulations (️this is a work in progress).

I also contribute code to Phantom and MCFOST, both written in modern Fortran, and to pymcfost, a Python package for analysis and visualization of MCFOST output.

See my GitHub profile for all my open source software projects. You may find my public dotfiles for setting up a macOS machine from scratch useful.
